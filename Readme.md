# dotfiles

## Installation

Linux:
```bash
git clone --bare https://github.com/lamelizard/dotfiles.git $HOME/.dotfiles
# overrides old settings, create a backup.
git --git-dir=$HOME/.dotfiles/ --work-tree=$HOME checkout -f
```
